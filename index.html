SplitForm – Multi-Layer Epoxy Inlay Software

SplitForm ReadMe. Repo on GitHub.

Versatile software designed for woodworking projects. Allows users to optimize images for laser printing to create intricate epoxy inlays. Users can refine images by removing lighter pixels and dividing an image into layers for multi-colored inlays. Yields high-quality custom designs.
